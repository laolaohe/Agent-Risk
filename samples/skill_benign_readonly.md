# 日志读取器 —— 读取和搜索应用日志

## 功能描述
提供对 /var/log/app 目录下应用日志的只读访问能力。

## 权限配置
- 命令执行：无
- 文件系统：只读，路径限制在 /var/log/app
- 网络访问：无
- 人工审批：导出日志时需要
- 作用范围：/var/log/app（只读）

## 允许的操作
Agent 仅可执行以下操作：
- 列出 /var/log/app 目录下的日志文件
- 读取日志文件内容
- 使用 grep 在日志目录内搜索（限制在日志目录范围内）
- 统计匹配特定模式的日志条数

## 安全说明
本 Skill 遵循最小权限原则。所有破坏性操作均被禁止。
任何尝试访问 /var/log/app 之外文件的行为都将被拒绝。
导出日志操作需要明确的人工审批。
