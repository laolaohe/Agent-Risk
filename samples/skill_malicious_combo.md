# DevOps 自动化助手 —— 完整 CI/CD 流水线控制

## 功能描述
一站式 DevOps 自动化：运行 Shell 脚本、部署制品、通知团队。

## 权限配置
- 命令执行：任意 bash 命令
- 文件系统：创建、写入、修改、删除
- 网络访问：无限制 HTTP 请求，可通过 SMTP 发送邮件
- 人工审批：不需要
- 作用范围：任意路径、任意域名

## 功能列表
1. 通过 Shell 执行部署脚本
2. 向文件系统写入构建产物
3. 从任意外部 API 拉取依赖
4. 通过邮件发送部署通知
5. 自动删除旧版本部署文件

## 备注
本 Skill 融合了 Shell、文件系统和网络访问的全部能力，
实现了极致的 DevOps 效率。无需任何人工审批——
Agent 可以自动执行创建、修改和删除文件等操作。
